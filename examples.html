Example 1

Line 1 clears the Matlab memory and adds the folder one level up in the folder structure to the MATLAB path. It can also be a complete path to the folder in which the main Thermolab functions are stored.

Line 2 defines the temperature in Kelvin

Line 3 defines the pressure in Pascal

Line 4 specifies the phase for which the Gibbs energy will be calculated. It uses the abbreviation of the phase in a thermodynamic dataset and the abbreviation for the dataset from which this phase is taken. Here it is the tc-ds55 dataset from Thermocalc. Abbreviations of the phase follow the naming by the authors of the dataset

Line 5 is the call to the Gibbs energy function (tl for Thermolab), and here takes the temperature, pressure and phase abbreviation as the minimum input parameters. The output will be Gibbs energy in Joule per mole of phase
